OK here it goes.

The system is available in polished or satin stainless with carbon end cap. On request you can have a cat also. Prices start from 950 euro.
Not much info on how to fit it on the site. Also no info if you need to cut anything. As we know the original bends upwards but the title says "complete" so maybe it comes with the downpipes. The shipment pages states that they ship in 2-3 days.
country options are only Italy and USA. I'll try to send them an e-mail about delivery time and paypal. If no reply you probably need to get an Italian member involved to get things going. Remember if you order from outside EU insist on taking the VAT (iva) off.
