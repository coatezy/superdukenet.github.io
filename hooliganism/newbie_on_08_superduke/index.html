The frame number includes the model year date towards the end. Year of registration is not necessarily the model year.

It only matters where there are model changes of course, but even if there weren't any that year, it will affect resale value with smart buyers, so pay for the model year, not the reg year.

If you are buying a standard SD; big model changes came in 07 (new headlight, tank, geometry, chassis part colours) smaller changes in paint, etc from 08, and again in 10.

If you are buying an SDR; big changes came in 08 with more power. Don't pay 08 money for an 07 SDR whatever the mileage.
