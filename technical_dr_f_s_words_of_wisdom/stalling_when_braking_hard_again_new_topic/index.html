Hi Superdukers,

Problem is finally (simply) solved due to a new fuelfilter set in the fuelpump (30,000km on the clock).

The foam blocks in the tank did not help.

Installing a 2009 tank and turning my 2005 fuelpump 180° did not help either.

Adjusting the sidepanels from an '05 tank to an '07 tank was easy piece.

So as already said by Reb, this stalling issue can be solved by changing the "expensive" fuel filters on the fuelpump.

Thanks for all the help dukes!
