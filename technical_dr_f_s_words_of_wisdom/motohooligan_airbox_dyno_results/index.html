Motohooligan airbox dyno results

Motohooligan

2008-11-13 01:14:00 UTC

So here it is.... 4hours of dyno work and air box swapin.

File 004-purple, Stock airbox with KTM paper filter.

File 008-Green, Stock box with BMC filter.

File 012-Red, Mar box with FRAM filter.

File 017-Blue, Moto box with K&N filter.

Stock air filter and BMC where run on the same PCIII map after remaping for the stock filter. There was very little, if any difference in the air/fuel graphs. And as you can see it did help a bit between 7200-8200RPM.
After fitting the mar unit it was remaped again as it went lean between 5000-8000RPM. I tried the mar unit with the K&N and the curves where near identical, with another small gain in the 7-8k area. After fitting the Moto box we remaped with the K&N filter. After the map we put the fram filter wuch mada a near identical pull with a small decrease between 6&8.5k. All the graphs shown are the average of 4 pulls. The bike was left straped to the dyno and the temp varied only 1.5degF the bar pressure and humitity did not change thruout the day.
