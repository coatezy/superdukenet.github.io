Not that I know anything on this but isn't there some limit over which there isn't any real reason to make it more tight?

Basically the front wheel is supposed to have little life in it and the damper should only take the biggest wiggles out. Example Keith Code said that the damper is reasanoble in almost the lowest setting where it doesn't affect the bike handling but will take the biggest moves off. Other "damping" is done by being relaxed on the bike.

And if the bike is extremely unstable then perhaps the problem might be somewhere else, like in suspension?
